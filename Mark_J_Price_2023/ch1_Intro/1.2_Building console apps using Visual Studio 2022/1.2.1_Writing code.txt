1.  Start Visual Studio 2022. 
    
    In the "Get started" section, click "Create a new project".


2.  on "all language" box, choose " C# ".

    then choose "console App".
    (making sure that you have chosen the cross-platform project template not the one for .NET Framework 
    which is Windows-only)

    then click Next.


3.  Then configure your project name (e.g : "first_project") and folder (e.g: "D/folder"). 
    click Next

4.  Then choose ".Net10.0 Long Term Support". 
    Check box labeled "Do not use top-level statements"
    
    click Create Project


5.  On "Solution Explorer" (right pane), click "Program.cs", which contain "hello World" project.

    (If you cannot see "Solution Explorer", then navigate to "View | Solution Explorer".)
